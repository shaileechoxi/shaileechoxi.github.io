Balade

Balade is an android application which books cab of a nearest available driver for a customer with the help of Google API calculating the longitude and latitude of customer as well as the driver. Basically it contains two system user a customer and a driver. It has two login and registration activity for customer and driver.

JAVA | Firebase | GoogleAPI | ANDROID Studio Software
