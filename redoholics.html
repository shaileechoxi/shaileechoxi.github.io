Redoholics - CRUD Web Application

Redoholics is a web application which allows user to store their selected item in favorites with edit update and delete functionality(CRUD).

  • Generated a service that allows access to API. It has the ability to get a single item, get the list of items, and create a new item along with delete and update functionality.
  • Protected routes/endpoints with JWT (JSON Web Tokens) and Extensive API testing with Postman.
  • Designed different components to make reusable blocks of code in front-end.
  • Includes a form and Angular Routing with Parameter.

Technologies-

Node.js | Express.js | MongoDB | Angular | Bootstrap
