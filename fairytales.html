Fairy-Tales

Fairy-Tales is an ecommerce website made on wordpress and core php with customise feedback plugin form. Woo-commerce plugin is used for products, cart, checkout pages with paypal integration. This website sells bridemaids attire along with wedding gowns

  • The site has a child theme designed with the home page which is designed from scratch and hand-coded CSS.
  • I have installed the woo-commerce plugin in the project to make this site fully functional as well as implemented papal and Stripe Payment Gateway.
  • To add more features the site has Coupon codes as well as tax rates for Ontario and Outside Ontario shipping.
  • Not only this but, it has customized feedback and subscribe now plugin form on the Home page of the site.

Technologies-

PHP | MYSQL(phpmyadmin) | HTML | CSS | WORDPRESS
