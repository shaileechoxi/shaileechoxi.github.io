CookieCoder

Cookiecoder is mainly a web application which is made in concern to developers who can create a developer profile/portfolio, share posts and get help from other developers easily. The project is build using React Hooks, Async/Await and better overall code.

  • Builded an extensive backend API with Node.js & Express
  • Protected routes/endpoints with JWT (JSON Web Tokens) and Extensive API testing with Postman.
  • Integrated React with backend in an elegant way, created a smooth workflow with API
  • Used Redux for app state management.
  • Along with that Created many container components that are integrated with Redux. Testing with the Redux Chrome extension

Technologies-

Node.js | Express.js | MongoDB | React | Redux | ES6+
